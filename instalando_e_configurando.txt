// - Instalação - //
Dependendo do sistema operacional, existe a possibilidade do GIT ja estar instalado. 
(MAC e Linux)
Caso o computador não tenha o GIT instalado, basta ir ao site oficial:
https://git-scm.com/downloads


// - Configurando o nome de usuário e o seu email. - //
git config --global user.name "Seu nome"
git config --global user.email "Seu email"
("--global" irá deixar essa configuração padrão para todos os projetos.)
